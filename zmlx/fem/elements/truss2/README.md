# Truss2 杆单元

二维杆单元 (2D Truss Element)，只承受轴向力。

## 基本信息

- **节点数**: 2
- **自由度**: 4（每个节点2个平动自由度 u, v）
- **应变状态**: 轴向工程应变（标量）
- **适用场景**: 桁架结构分析（塔吊、屋架、网架、桥梁桁架等）

## 文件说明

| 文件 | 功能 |
|------|------|
| `_stiffness.py` | 计算单元刚度矩阵（4×4） |
| `_strain.py` | 计算轴向工程应变（标量）：ε = (L_new - L0) / L0 |
| `_stress.py` | 计算轴向应力（标量）：σ = E · ε |
