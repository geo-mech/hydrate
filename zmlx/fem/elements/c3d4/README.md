# C3D4 四面体单元

三维常应变四面体单元 (Constant Strain Tetrahedron)。

## 基本信息

- **节点数**: 4
- **自由度**: 12（每个节点3个平动自由度 u, v, w）
- **应变状态**: 常应变（单元内应变为常量）
- **适用场景**: 三维实体结构的应力分析，适合复杂几何的网格划分

## 文件说明

| 文件 | 功能 |
|------|------|
| `_stiffness.py` | 计算单元刚度矩阵（12×12） |
| `_strain.py` | 计算单元应变向量 (6,)：[ε_xx, ε_yy, ε_zz, γ_xy, γ_yz, γ_zx] |
| `_stress.py` | 计算单元应力向量 (6,)：[σ_xx, σ_yy, σ_zz, τ_xy, τ_yz, τ_zx] |
