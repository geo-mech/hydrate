# C3D8 六面体单元

三维八节点六面体单元 (8-Node Hexahedron)，采用 2×2×2 高斯积分。

## 基本信息

- **节点数**: 8
- **自由度**: 24（每个节点3个平动自由度 u, v, w）
- **应变状态**: 非常应变（应变在单元内变化，通过高斯积分点计算）
- **适用场景**: 三维实体结构的应力分析，适合规则几何形状，精度高于四面体单元

## 文件说明

| 文件 | 功能 |
|------|------|
| `_stiffness.py` | 计算单元刚度矩阵（24×24），包含形函数及其导数的辅助函数 |
| `_strain.py` | 计算单元应变，支持单点、高斯积分点及节点外推三种方式 |
| `_stress.py` | 计算单元在指定自然坐标处的应力向量 (6,) |
