# Planar Strain Quad4 平面应变四边形单元

平面应变双线性四边形单元 (4-Node Quadrilateral, Plane Strain)，采用 2×2 高斯积分。

## 基本信息

- **节点数**: 4
- **自由度**: 8（每个节点2个平动自由度 u, v）
- **应变状态**: 非常应变（应变在单元内变化）
- **适用场景**: 厚度方向尺寸远大于面内尺寸的结构，比三角形单元精度更高

## 文件说明

| 文件 | 功能 |
|------|------|
| `_stiffness.py` | 计算单元刚度矩阵（8×8） |
| `_strain.py` | 计算单元在指定自然坐标处的应变向量 (3,) |
| `_stress.py` | 计算单元在指定自然坐标处的应力向量 (3,) |


## 平面应变 D 矩阵

$$D = \frac{E}{(1+\mu)(1-2\mu)} \begin{bmatrix} 1-\mu & \mu & 0 \\ \mu & 1-\mu & 0 \\ 0 & 0 & \frac{1-2\mu}{2} \end{bmatrix}$$
